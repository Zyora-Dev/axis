Reading the results honestly

  • Axis's win is memory. The frozen 7B base carries no optimizer state and stays in bf16, so Axis fine-tunes in 17% less GPU memory (23.3 GB vs 28.2 GB) — headroom for longer context or larger batches on the same card.
  • PyTorch is faster here (~15%). A decade of hand-tuned CUDA kernels and PEFT integration is ahead on raw LoRA throughput at 7B. We publish that, not just the wins.
  • Apples-to-apples. Both train the same ~40M adapters on the same 7 projections (r=16), same Llama-2-7B shape, same bf16 + flash attention, each in its own fresh container.
  • Full-model 7B training needs ~112 GB for bf16 weights + grads + fp32 AdamW state — it doesn't fit on one 80 GB GPU for either framework; that's a multi-GPU (ZeRO / FSDP) benchmark, run separately.
